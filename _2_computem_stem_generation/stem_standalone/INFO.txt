INCOSTEM - STEM Simulation Source Code
=======================================

C++ Source Files Included:
- incostemcmd.cpp (main program)
- incostem.cpp/hpp (core calculation)
- slicelib.cpp/hpp (scattering library)
- cfpix.cpp/hpp (FFT handler)
- floatTIFF.cpp/hpp (TIFF file I/O)
- probe.cpp/hpp (probe generation)
- ransubs.cpp/hpp (random numbers)
- rfpix.cpp/hpp (real image handler)
- psave.cpp/hpp (save utilities)
- atompot.cpp (atomic potentials)
- fftw3.h, newD.hpp (headers)

Sample Input Files:
- graphene.xyz, SI100.XYZ, si110.xyz, si3n4.xyz, SRT.XYZ
- Plus .DAT data files

To Compile (example):
g++ -o incostem incostemcmd.cpp incostem.cpp probe.cpp slicelib.cpp floatTIFF.cpp cfpix.cpp ransubs.cpp rfpix.cpp atompot.cpp -lfftw3f

Required Library:
- FFTW3 (libfftw3f)

Program Usage:
./incostem
(Then enter parameters when prompted)

Input file format: XYZ atomic coordinates
Output: TIFF image file
